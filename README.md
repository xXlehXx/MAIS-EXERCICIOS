# 70 Exercícios de Lógica e Estrutura de Dados em Java ☕

Este repositório contém a resolução de **70 exercícios** de programação desenvolvidos durante a monitoria da faculdade (**CEUB**). O foco principal foi o fortalecimento da lógica de programação, manipulação de vetores, matrizes e funções.

## 🚀 Tecnologias Utilizadas
* **Java**: Linguagem principal para resolução dos problemas.
* **Scanner**: Utilizado para entrada de dados via console.

## 📌 Conteúdos Abordados
A lista evolui em complexidade, cobrindo os seguintes tópicos:
* Estruturas de Controle (if/else, switch)
* Laços de Repetição (for, while)
* Manipulação de Vetores (Busca, remoção, compactação)
* Matrizes (Transposição, identidade, busca por linha)
* Algoritmos de Ordenação (Bubble Sort)
* Cálculos Matemáticos (Fibonacci, Área de Circunferência)

## 📂 Estrutura de Arquivos
Cada arquivo segue a nomenclatura `exXX.java`, onde `XX` representa o número do exercício na lista original da monitoria.

## 💻 Como executar
Certifique-se de ter o JDK instalado em sua máquina.
1. Compile o arquivo:
   ```bash
   javac ex01.java
2. Execute:
   ```bash
   java ex01
